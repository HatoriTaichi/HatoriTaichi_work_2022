// ルール
オンライン対戦FPSゲームで、相手を撃って相手のストックを無くしていくゲームです。
相手のストックが「0」になれば勝ちです。

// 操作方法
WASD : 移動
マウス : 視点移動
左クリック : 射撃

プログラム全部
